ddu file

Dodeca allows to view and modify ddu files. You can open ddu shipped with Dodeca via button on bottom bar. If you already have a ddu, locate it with file manager and choose to open with Dodeca. Dodeca will import this ddu to internal folder.

Motion

You can change position of ddu by dragging it. You can zoom/scale ddu with 2 fingers. Note, that if Settings/Redraw trace is on, motion and scaling clear canvas.

Bottom bar

  • Tap on area above bottom bar to toggle it.
  • Long tap on button to see brief description.
  • Bottom bar hides after 20 seconds of inactivity.
