Выбирая ddu-картину

При первом открытии колекции ddu-картин вы увидите загрузающиеся предпросмотры ― они автогенерируются на основе имеющихся файлов (не скачиваются из Интернета). Додека Медитация ― Android версия видеоарт-программы DodecaLook для Windows. Если вы обнаружили ошибку или у вас есть предложения по улучшению ― зайдите в Настройки/Обратная связь и поддержка и опишите это в письме. Свободно для некоммерческого использования.

ddu-картина

Додека Медитация позволяет просматривать и изменять ddu-картины. Вы можете открыть ddu-картину, из коллекции Додека Медитация с помощью кнопки на нижней панели. Если у вас уже есть файл ddu-картины, найдите его в любом файловом менеджере и выберите Открыть с помощью/Додека Медитация. Додека Медитация автоматически импортирует файл этой ddu-картины во внутреннюю папку. Файл ddu-картин (с расширением .ddu) содержит исходные данные картины и правила её развития.

Движения

Вы можете передвигать ddu-картину перетаскиванием.
Вы можете масштабировать ddu-картину двумя пальцами.
Если включена настройка Перерисовка следа, то при движении холст будет очищаться.
