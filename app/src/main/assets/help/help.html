ddu-picture

Dodeca Meditation allows to view and modify ddu-pictures. You can open a ddu-picture from Dodeca Meditation art collection via button on bottom bar. If you already have a file of ddu-picture, locate it with file manager and choose to open with Dodeca Meditation. Dodeca Meditation will import file of the ddu-picture to internal folder. File of ddu-picture (with extension .ddu) contains initial state and development rules of the picture.

Motion

You can change position of ddu-picture by dragging it.
You can zoom/scale ddu-picture with 2 fingers (pinch to zoom).
Note, that if Settings/Redraw trace is on, motion and scaling clear canvas.
