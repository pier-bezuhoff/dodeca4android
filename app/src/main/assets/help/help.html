General info

Dodeca Meditation has a Windows predecessor: DodecaLook, an older app for generative video art of aesthetic geometry (you can find it in the app description on Google Play). If you found a bug or if you want to suggest some improvements ― go to Settings/Feedback and support to email it to me. The app and its are free for non-commercial use.

ddu-pictures

Dodeca Meditation allows you to watch and modify ddu-pictures. You can look through ddu-pictures provided for Dodeca Meditation art gallery via the button on the bottom bar. If you already have a *.ddu file, you can import it from the gallery or locate it with a file manager and choose to open with Dodeca Meditation. Dodeca Meditation will import the file to its internal folder. The file of a ddu-picture (with extension .ddu) contains initial state and development rules of the picture.

Moving around

You can change the position of a ddu-picture by dragging it.
You can zoom/scale ddu-pictures with 2 fingers (pinch to zoom).
Note, that if Settings/Redraw trace is on, any motion and scaling will clear the canvas.
