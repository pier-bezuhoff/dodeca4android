ddu file

Dodeca Look allows to view and modify ddu files. You can open ddu shipped with Dodeca Look via button on bottom bar. If you already have a ddu, locate it with file manager and choose to open with Dodeca Look. Dodeca Look will import this ddu to internal folder.

Motion

You can change position of ddu image by dragging it. You can zoom/scale ddu image with 2 fingers (pinch to zoom). Note, that if Settings/Redraw trace is on, motion and scaling clear canvas.
